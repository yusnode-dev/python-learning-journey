## 🗺️ ФАЗЫ ОБУЧЕНИЯ

---

### ФАЗА 1 — Фундамент (Месяц 1–2)
**Цель:** Уверенно писать Python код, понимать как он работает

#### Что изучаю:
- [ ] Переменные, типы данных (int, str, float, bool, None)
- [ ] Условия (if / elif / else)
- [ ] Циклы (for / while / break / continue)
- [ ] Функции (def, return, args, kwargs)
- [ ] Исключения (try / except / finally)
- [ ] Работа со списками, словарями, множествами
- [ ] Работа с файлами (open, read, write)
- [ ] Модули и импорты
- [ ] Git: commit, push, branch, merge

#### Проект фазы:
> CLI-приложение: менеджер задач (добавить/удалить/показать задачи, сохранение в файл)

#### Финансовая цель:
- Доход: ~$370/мес (текущая работа)
- Накопить за фазу: $200–400

---

### ФАЗА 2 — OOP + первый реальный проект (Месяц 3–5)
**Цель:** Мыслить как инженер, проектировать системы

#### Что изучаю:
- [ ] Классы и объекты
- [ ] Инкапсуляция
- [ ] Наследование
- [ ] Полиморфизм
- [ ] Абстракция
- [ ] Composition vs Inheritance
- [ ] SOLID принципы
- [ ] Декораторы
- [ ] Датаклассы
- [ ] Базовые паттерны (Factory, Strategy)

#### Проект фазы:
> Telegram-бот: простой бот с командами, FSM, сохранение данных

#### Финансовая цель:
- Доход: ~$370/мес
- Накопить за фазу: $400–700 (итого ~$600–1100)

---

### ФАЗА 3 — Backend: FastAPI + PostgreSQL (Месяц 6–9)
**Цель:** Строить production-ready API, работать с базами данных

#### Что изучаю:
- [ ] FastAPI: роуты, модели, валидация
- [ ] REST API принципы
- [ ] Middleware, обработка ошибок
- [ ] JWT аутентификация
- [ ] PostgreSQL: основы, запросы
- [ ] SQL: SELECT, JOIN, GROUP BY, индексы
- [ ] SQLAlchemy ORM
- [ ] Миграции (Alembic)
- [ ] Docker basics
- [ ] Переменные окружения (.env)

#### Проект фазы:
> Backend API для Telegram-бота: пользователи, авторизация, данные в PostgreSQL

#### Финансовая цель:
- Доход: $400–700/мес (первые фриланс-заказы)
- Накопить за фазу: $1500–2500 (итого ~$2000–3500)

---

### ФАЗА 4 — AI Systems + рост (Месяц 10–14)
**Цель:** Строить AI-продукты, зарабатывать на них

#### Что изучаю:
- [ ] Async Python (asyncio, await, event loop)
- [ ] Async FastAPI
- [ ] LLM интеграция (OpenAI API, Anthropic API)
- [ ] Tool calling
- [ ] Memory systems
- [ ] pgvector (векторные базы данных)
- [ ] Agent loops
- [ ] aiogram 3 advanced (FSM, middleware, routers)
- [ ] Redis (кэширование)
- [ ] Background tasks

#### Проект фазы:
> AI Telegram-бот с памятью: принимает вопросы, отвечает через LLM, хранит историю

#### Финансовая цель:
- Доход: $800–1500/мес
- Накопить за фазу: $5000–8000 (итого ~$7000–11500)

---

### ФАЗА 5 — Senior уровень + $3k цель (Месяц 15–18)
**Цель:** Портфолио из 5+ проектов, remote job или стабильный фриланс

#### Что изучаю:
- [ ] System Design основы
- [ ] Масштабирование систем
- [ ] Message queues (Celery, RabbitMQ)
- [ ] Мониторинг и логирование
- [ ] CI/CD pipelines
- [ ] Безопасность API
- [ ] Оптимизация запросов
- [ ] Архитектурные паттерны

#### Проект фазы:
> SaaS продукт или сложный AI-backend: полноценная система с деплоем

#### Финансовая цель:
- Доход: $2000–3000+/мес
- Накоплено итого: $15,000–20,000+
